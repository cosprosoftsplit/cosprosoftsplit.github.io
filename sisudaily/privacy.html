SisuDaily Privacy

SisuDaily is a private reflection journal. It stores your journal entries, prompt completions, and path progress locally on this device only.

SisuDaily does not collect, transmit, sync, or share any personal data with any server. There is no account. There is no analytics SDK. There is no third-party tracking.

StoreKit purchases are handled by Apple. SisuDaily only receives the entitlement state.

You can delete all SisuDaily data at any time from Settings → Data → Delete all local data.
